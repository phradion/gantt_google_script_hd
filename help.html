Features

  • Calendar bar renders automatically based on START DATE
  • Task's End date calculated automatically
  • Weekends are excluded from tasks automatically
  • Parent task's start and end dates are calculated automatically based on children tasks.
  • Detecting Parent task based on different background color
  • Percentage % of Tasks are showned in days
  • Japanese and Vietnamese Holidays and observances render automatically
  • PMO and Delivery dates render automatically
  • Task Overdue alert

How to use

Calendar
  • Only Enter data the into the bright yellow fields
  • Change Start Date to your project's start date to update the calendar bar (Months and Days) automatically
  • Change PMO dates (1,2,3,4,5,6) and Delivery Date on row 3 to render them in the chart automatically
  • Holidays are rendered automatically everytime you open this file based on data in our database.
Tasks and Chart
  • Task's end date calculated automatically by entering start date and duration
  • Task's progress are displayed in the chart based on the percentage %. Valid value is between 0% and 100%
  • Task with NONE-WHITE background are assumed parent tasks and all the white background tasks beneath it are considered its children tasks.
  • field R1 R2 are placeholders to customise for your project
  • Duration is always rounded up. Duration like 0.5 is treated as 1. Duration like 1.5 treated as 2.
